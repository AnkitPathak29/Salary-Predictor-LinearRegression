🧾Salary Predictor using Linear Regression– Readme
-
📌Project Overview
-
This project is a simple Machine Learning application built using Python and Linear Regression from ‘scikit-learn’.  
The program predicts a person's salary based on their *years of experience using a dataset named ‘Salary_Data.csv’.


It also:
- trains a linear regression model,
- displays the regression equation,
- takes user input for experience,
- predicts salary,
- and visualizes the dataset with the best-fit regression line.
  

🎯Objective
-
The main objective of this project is to understand how Linear Regression works in a real-world scenario by predicting salary from years of experience.  

It helps in learning:
- dataset handling using pandas.
- numerical operations using NumPy.
- model creation using scikit-learn.
- data visualization using matplotlib.


 🧪Technologies Used
 -

- NumPy
- Pandas

- Scikit-learn
